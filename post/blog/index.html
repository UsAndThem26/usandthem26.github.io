如何使用Hugo搭建个人博客网站【Mac端】 @ UsAndThem26 | Saturday, Jan 15, 2022 | 2 | Saturday, Jan 15, 2022

【打开终端】

一、使用brew安装Hugo

     * 需要提前下载安装brew，网上有详细教程。

 brew install hugo

 hugo version

     * 查看Hugo版本，安装完成。

二、使用Hugo生成一个博客网站

 hugo new site myblog

     * Hugo为你创建了一个myblog文件夹，你网站的所有数据都将储存在这个文件夹当中。 

     示例：
     “Congratulations! Your new Hugo site is created in /Users/juanjuanli/mybolog.
     ...“

三、为网站设置一个主题

     * 可以到Hugo主题官网：themes.gohugo.io下载。

     * 根据提示安装和配置主题，例如我选择了一个叫“Dream”的主题。


 1.下载主题

     * 使用git克隆主题到本地。
     
     * 需要确认提前装好git，网上同样有详细教程，非常简单。

 跟目录下

 cd themes

 git clone https://github.com/g1eny0ung/hugo-theme-dream.git dream

     示例：
     "正克隆到 'dream'...
     ...
     接收对象中: 100% (3111/3111), 20.08 MiB | 64.00 KiB/s, 完成.
     处理 delta 中: 100% (1759/1759), 完成."


 2.配置主题

     1）修改配置文件：拷贝myblog/themes/dream/exampleSite/config.toml到根目录替换原有config.toml

     2）打开替换好的config.toml并编辑theme = "dream"，保存。        
  
     3）其他参数根据自己的意愿修改。

         例如：
         baseurl = "https://usandthem26.github.io"
         languageCode = "en"
         defaultContentLanguage = "zh-cn"
         title = "UsAndThem26的博客"
         theme = "dream"
         ...
         author = "UsAndThem26"
         # description = "UsAndThem26的碎碎念"
         avatar = "/img/avatar.jpeg"
         headerTitle = "UsAndThem26的博客"
         motto = "向没有迷茫和沉沦在有限事物里的大脑们致敬"
         ...

四、在本地启动个人博客

 根目录下 
 
 hugo server -t dream --buildDrafts

     * 得到本地网站例如：http://localhost:1313/usandthem26

     示例：
     "Start building sites … 
     hugo v0.91.2+extended darwin/arm64 BuildDate=unknown
     ...
      Running in Fast Render Mode. For full rebuilds on change: hugo server --disableFastRender
      Web Server is available at http://localhost:1313/ (bind address 127.0.0.1)
      Press Ctrl+C to stop"

     * 键入Ctrl+C结束服务后再更改配置内容需要再hugo server -t dream --buildDrafts方可查看更改后的内容。

     * 键入Ctrl+C结束前自动刷新，即更即看。

五、写一篇文章

 1.根目录下 
 
 hugo new post/blog.md

     * 生成文件位置位于myblog/content/post目录下。


 2.使用visual studio code打开位于myblog/content/post的blog.md

     * 更改标题并写一篇博客，保存。

     示例：
     ”---
     title: "如何使用Hugo搭建个人博客【Mac端】"
     date: 2022-01-15T18:06:08+08:00
     draft: true
     ---“
     ...


 3.根目录下再次 
 
 hugo server -t dream --buildDrafts

     * 刷新网页，博文已更新到博客网站。

六、将个人博客部署到远端服务器

     * 部署到GitHub需要提前拥有一个GitHub账号，登入并创建一个储存库。

     * 储存库名称需要与GitHub账户名保持一致。

 1.将个人博客网站部署到新建的GitHub储存库
 
 根目录下 
 
 hugo --theme=dream --baseUrl="https://usandthem26.github.io/" --buildDrafts

     * 生成一个public文件，里面有你的博客内容，然后再把public文件夹上传到GitHub储存库。

 
 2.

 根目录下

 cd public

 git init

     “已重新初始化已存在的 Git 仓库于 /Users/juanjuanli/myblog/public/.git/”

 git add .

 git commit -m "第一次提交"

     * 写一个提交信息

     示例：
     "git commit -m "第一次提交"[master（根提交） 7346f86]第一次提交
     31 files changed, 2069 insertions(+)
     create mode 100644 404.html
     create mode 100644 categories/index.html
     ..."


 3.将本地public文件夹跟远端GitHub仓库进行关联

 git remote add origin https://github.com/usandthem26/usandthem26.github.io.git

 
 4.推到远端

 git push -u origin master

     * 输入用户名和密码

     * 如果输入密码这步提示需要个人令牌，需要到GitHub生成一个个人令牌，网上也有详细教程，很简单。
     
     示例：
     “枚举对象中: 45, 完成.
     对象计数中: 100% (45/45), 完成.
     使用 8 个线程进行压缩
     压缩对象中: 100% (40/40), 完成.
     写入对象中: 100% (45/45), 297.36 KiB | 27.03 MiB/s, 完成.
     总共 45（差异 10），复用 0（差异 0），包复用 0
     remote: Resolving deltas: 100% (10/10), done.
     To https://github.com/usandthem26/usandthem26.github.io.git
     * [new branch]      master -> master
     分支 'master' 设置为跟踪来自 'origin' 的远程分支 'master'。”

 5.成功部署到GitHub。

七、成功拥有一个博客网站 https://usandthem26.github.io/
