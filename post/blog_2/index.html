如何更新文章到博客 @ UsAndThem26 | Sunday, Jan 16, 2022 | 1 | Sunday, Jan 16, 2022 

     * 使用Hugo更新文章到博客

     * 原来更新博文很简单，超级简单，几乎就是重复写入命令，看看能不能找个脚本省去这些重复的步骤吧。

     * 就看我有没有文章可以更新了。

【打开终端】

一、根目录下新建一个.md文件

 hugo new post/blog_1.md

     * 生成文件位置位于myblog/content/post目录下。
     
     示例：
     “Content "/Users/juanjuanli/myblog/content/post/blog_1.md" created
     ...“

二、启动本地个人博客

 根目录下

 hugo server -t dream --buildDrafts

     示例：
     ”Start building sites … 
     …
     Running in Fast Render Mode. For full rebuilds on change: hugo server --disableFastRender
     Web Server is available at http://localhost:1313/ (bind address 127.0.0.1)
     Press Ctrl+C to stop“
 
 ^C% 

 2.使用visual studio code打开位于myblog/content/post的blog_1.md

     * 更改标题并写一篇博客，保存。

三、部署到新建的GitHub储存库

 根目录下 

 hugo --theme=dream --baseUrl="https://usandthem26.github.io/" --buildDrafts


     “Start building sites … 
     hugo v0.91.2+extended darwin/arm64 BuildDate=unknown

     Total in 27 ms”
 


 根目录下

 cd public

 git init

     “已重新初始化已存在的 Git 仓库于 /Users/juanjuanli/myblog/public/.git/
     juanjuanli@JuanjuandeAir public %” 

 git add .

 git commit -m "第二次提交"

     [master 2aa315c] 第二次提交
     Committer: ...

     8 files changed, 569 insertions(+), 120 deletions(-)
     create mode 100644 post/blog_1/index.html

 git remote add origin https://github.com/usandthem26/usandthem26.github.io.git


     “error: 远程 origin 已经存在。”


 git push -u origin master

     “枚举对象中: 25, 完成.
     ...
     remote: Resolving deltas: 100% (9/9), completed with 5 local objects.
     To https://github.com/usandthem26/usandthem26.github.io.git
     1cc37ab..2aa315c  master -> master"
     分支 'master' 设置为跟踪来自 'origin' 的远程分支 'master'。"

     * 刷新网页文章未更新到博客。

继续

 git remote add origin https://github.com/usandthem26/usandthem26.github.io.git
     
     "error: 远程 origin 已经存在。

 git push -u origin master

     “分支 'master' 设置为跟踪来自 'origin' 的远程分支 'master'。
     Everything up-to-date”
    
     * 博文更新完成

五、刷新网页https://usandthem26.github.io/

六、博客已更新
